# Affix-Lib
This "Library" implements affix like in bootstrap3. Unfortunately they removed it from bootstrap4!

## How to apply it!
```html
<html>
  <head>
    <script src="affix-lib.js"></script>
  </head>
  
  <body>
    <!-- You just use affix like in bootstrap3 -->
    <div data-spy="affix">
      <!-- ... -->
    </div>
  </body>
</html> 
```
